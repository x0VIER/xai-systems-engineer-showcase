Distributed Key-Value Store

Raft Consensus

Implements the Raft consensus algorithm for leader election and data replication, ensuring consistency across nodes.

gRPC Communication

Uses gRPC for efficient, type-safe communication between nodes and clients with Protocol Buffers.

Fault Tolerance

Maintains availability and consistency even when nodes fail, with automatic leader election.

Interactive GUI

Provides a simple GUI for interacting with the store, built with egui for a native look and feel.
