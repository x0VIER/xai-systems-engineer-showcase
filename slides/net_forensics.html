Network Forensics CLI

Packet Capture

Uses libpcap to capture network packets from any interface, with configurable packet limits.

Protocol Analysis

Parses Ethernet, IPv4, IPv6, TCP, UDP, and other protocols using the pnet crate for deep packet inspection.

Data Visualization

Generates protocol distribution charts using the plotters crate, providing insights into network traffic patterns.

Command-Line Interface

Simple CLI with configurable options for interface selection and capture limits, built with the clap crate.
